Windows 10 dual boot with Ubuntu

#Contents
  1. Download Software Needed
  2. Create Live Ubuntu Pendrive
  3. Configure Dual Boot

#Download Software Needed

Download All softwares mentioned below.
Download Ubuntu
Download EasyBCD
Download YUMI Multiboot creator download suitable version(bios/uefei)



#Create Live Ubuntu Pendrive

1.Open Yumi

2.Insert USB

3.Select Pendrive

4.Select Ubuntu

5.Browse and select iso file

6.Create usb

After Ubuntu usb is created, install Ubuntu on one of your partition which will be turned into ext3/ext4 format.

#Configure Dual Boot

1.Update grub for recovering Windows
Now after successfull installation of Ubuntu your default bootloader will change from Windows bootloader to grub2.0. Windows 10 entry is automativally added in grub.
Restart computer and boot into Windows 10., Open EasyBCD and follow given steps:
Step 1: Click on Add New Entry on left bar
Step 2: Click on Linux tab on top
Step 3: Select Type: grub2, Name:xyz, Drive:Automatic Locate and Load
Step 4: Click on Add entry, then Save Setting.

Now you will boot into Windows bootloader where you will be asked to choose an OS. Enjoy your dual boot:)Thank you for your support.
